# Exam Date Booking App

A mobile application for managing and tracking exam dates.
Users can create, edit, and view upcoming exams with a clean and simple interface.

---

## Features
- Login system
- Home dashboard
- List of exams
- Create exam
- Edit exam
- Profile settings
- Bottom tab navigation (Home, Timer, Profile)
- Timer feature (planned)

---

## Tech Stack
- React Native
- Expo
- Expo Router
- JavaScript

---

1. Install Dependencies
```bash
cd client
npm install
```

2. Start
```bash
npm start
```
3. Install Dependencies
```bash
cd server  
npm install
```

4. Start
```bash
npm run start:dev
```

If you got Command not found: exp
```bash
npm install -g expo-cli
```
